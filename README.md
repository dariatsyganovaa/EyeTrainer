\# Eye Trainer



Приложение для тренировки зрения при офтальмологических заболеваниях.



\## Основные задачи:



\- Исследование особенностей восприятия цвета и формы при различных заболеваниях глаз

\- Алгоритм персонализации тренировки на основе диагноза пользователя

\- Qt-интерфейс приложения и интеграция модулей команды

\- База данных (PostgreSQL) — хранение пользователей, диагнозов, планов тренировок и истории результатов



\## Стек



Python, PyQt5, PostgreSQL



\## Установка и запуск



```bash

pip install -r requirements.txt

python run.py

```
